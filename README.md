# International Conference Web Engineering 2016 Mobile Web App
##Read ME
The purpose of this app is creating a web application for meetings and conventions. 
It is based on Node.js, express.js, Polymer and Dust.js. 
####Features 
####Modules
* `body-parser`
* `cookie-parser`
* `debug`
* `adaro`
* `dustjs-helpers`
* `dustjs-linkedin`
* `express`
* `morgan`
* `serve-favicon`
